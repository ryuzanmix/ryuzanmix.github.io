録音の基本知識

最重要
MIXの正しい理解 - 魔法ではありません

MIXは「録音された素材を最大限に活かし、楽曲として完成させる技術」です。
決して魔法ではなく、元の録音品質を超えた結果を生み出すことはできません。

MIXでできること ✨
  • 録音された歌声の魅力を最大限に引き出す
  • 音質を整え、聞きやすくする
  • 楽曲全体のバランスを調整する
  • 軽微なタイミングズレの修正
  • 自然なエコーやエフェクトの追加
  • ノイズの除去・軽減
MIXでできないこと 🚫
  • 歌唱力そのものを根本的に変える
  • 極端に音質の悪い録音を完全に修復
  • 大幅なタイミングズレを自然に修正
  • 存在しないハモリや歌声を作り出す
  • 録音されていない音を追加する
  • 感情表現や歌い方を変える

つまり、録音品質がMIXの仕上がりを決める

良い録音 × 良いMIX = 素晴らしい仕上がり
悪い録音 × 良いMIX = 限界のある仕上がり

だからこそ、このガイドの録音方法が重要なのです。 MIX師は皆さんの録音素材を最大限に活かすために全力を尽くしますが、 元の録音品質を超えた奇跡は起こせません。

💡 このガイドの録音方法を実践していただければ、MIXで十分に美しく仕上げることができます！

🎤 マイクとの距離・角度

✅

理想的な録音スタイル

  • 距離：マイクから20-30cm（握りこぶし2-3個分）
  • 角度：口とマイクが正面になるように
  • 高さ：口の高さに合わせる
  • 姿勢：まっすぐ立って、リラックス
❌

避けるべきNG録音

  • 近すぎ：5cm以内は息づかいやノイズが入る
  • 遠すぎ：50cm以上は声が小さく反響する
  • 横向き：斜めからの録音は音質が不安定
  • 手で覆う：スマホを手で覆うと音がこもる

⏰ タイミング・テンポの重要性

なぜタイミングが重要？

カラオケ音源と歌のタイミングがずれていると、MIX時に大幅な修正が必要になり、 自然な仕上がりが困難になります。一定のテンポで、音源に合わせて歌うことが最も重要です。

良いタイミング
  • カラオケ音源をしっかり聞きながら歌う
  • メトロノームを意識して一定のテンポ
  • 楽曲の「入り」を正確に合わせる
  • ブレス（息継ぎ）のタイミングも計算
悪いタイミング
  • 音源を聞かずに自分勝手なテンポで歌う
  • 走る（早くなる）・もたる（遅くなる）
  • 歌詞の出だしがバラバラ
  • 間奏や前奏を無視して歌い出す

🏠 録音環境の整え方

🏡

理想的な場所

  • 寝室：布団やカーテンが音を吸収
  • クローゼット：衣類が自然なリバーブを作る
  • 畳の部屋：畳が適度に音を吸収
  • カーペット敷きの部屋：足音や反響を軽減
🚫

避けるべき場所

  • お風呂・トイレ：反響が強すぎる
  • キッチン：冷蔵庫や換気扇の音
  • 窓際：外からの騒音が入る
  • エアコンの下：風切り音が録音される

🔊 音量バランスの調整

適切な音量で録音するコツ
🎵 普通の話し声の1.5倍程度
大きすぎず小さすぎず、自然な歌声で録音してください
⚠️ 音量を一定に保つ
サビで急に大きくなったり、Aメロで小さくなりすぎないよう注意
✅ 録音レベルをチェック
アプリの録音レベルメーターが赤くならない範囲で調整

✅ 録音後のチェックリスト

録音完了後に必ず確認してください！

以下の項目をチェックすることで、MIX時の修正を最小限に抑え、 より自然で美しい仕上がりを実現できます。

🎵 音質・音量チェック

  • ✅ 録音全体を通して聞いてみる
  • ✅ 自分自身が聞いて問題なく聞こえるか
  • ✅ 音量が一定に保たれているか
  • ✅ 極端に大きい・小さい部分がないか
  • ✅ 音割れ（クリップ）していないか
  • ✅ 背景ノイズが少ないか

⏰ タイミング・テンポチェック

  • ✅ カラオケ音源と同時再生してズレがないか
  • ✅ 歌い出しのタイミングが合っているか
  • ✅ サビ・Aメロなど各パートの入りが正確か
  • ✅ 走り・もたりがないか
  • ✅ ブレス（息継ぎ）が自然か
詳細チェックポイント
🎤 声質・歌い方チェック
• 自然な歌声になっているか（力みすぎていないか）
• 歌詞が聞き取りやすいか（滑舌は明瞭か）
• 感情表現が楽曲に合っているか
📁 ファイル管理チェック
• ファイル形式がWAVまたはAIFFか
• ファイル名が分かりやすいか
• 頭出し（無音部分のカット）ができているか
• 複数テイクがある場合は整理されているか
🎵 一気に歌いきれない場合
• Aメロ、Bメロ、サビなどパートごとに分けて録音OK
• 各パートの最初と最後に少し余裕を持たせる
• ファイル名で区別（例：rec_vo_verse1.wav、rec_vo_chorus.wav）
• どの部分をどのファイルに録音したかメモを残す
⚠️ 再録音が必要な場合
• 音量が極端に小さい/大きい
• 明らかなタイミングのズレ
• 背景ノイズが多すぎる
• 音割れが頻発している
• 自分で聞いて違和感がある
客観的な聞き返しのコツ
🎧 聞き返しの手順
1. 録音完了後、5-10分休憩を取る
2. ヘッドホンまたはイヤホンで全体を通して聞く
3. 「他の人が聞いたらどう思うか」を意識する
4. 気になる部分があれば時間を記録しておく
✅ 合格基準
• 自分が聞いて「これなら人に聞かせられる」と感じる
• 楽曲の世界観を損なわない自然な歌声
• 歌詞が明確に聞き取れる
• 全体的に安定した音質
🔄 パート別録音の例
• イントロ＋Aメロ → rec_vo_verse1.wav
• Bメロ → rec_vo_bridge.wav
• サビ → rec_vo_chorus.wav
• 2番Aメロ → rec_vo_verse2.wav
• 大サビ → rec_vo_final_chorus.wav
🎯

理想的な録音データ

  • 音質：クリア（背景ノイズ最小）
  • 音量：一定レベル（-12dB～-6dB程度）
  • タイミング：カラオケ音源と完全同期
  • 形式：WAV/AIFF（44.1kHz/24bit以上）
  • 頭出し：無音部分カット済み
🚨

要注意な録音データ

  • 音割れ：レベルオーバーで音が歪む
  • タイミングズレ：テンポが不安定
  • ノイズ混入：エアコン、車の音等
  • 音量変動：大きすぎる/小さすぎる部分
  • 圧縮形式：MP3等の音質劣化

チェック完了後の最終確認

すべての項目をチェックし、問題がなければ：

  1. ファイル名を適切にリネーム
  2. 不要なテイクを削除して整理
  3. アップロード用フォルダにまとめる
  4. 提出時のメモを準備（録音環境、使用機器、特記事項など）

💡 プロからのアドバイス

「完璧を目指さず、自然体で」
多少の息継ぎや小さなミスは後から修正できます。緊張しすぎず、 普段通りの歌声で録音してください。一番大切なのは、楽しみながら歌うことです😊

iPhone での録音方法

使用率 No.1
🎵

GarageBand

おすすめ★★★

無料・高音質・操作簡単で初心者に最適

  1. GarageBandを開く
  2. 「+」→「AUDIO RECORDER」を選択
  3. 楽曲を別のアプリで再生
  4. 赤い録音ボタンで録音開始
  5. 録音完了後「↓」→「書き出し」→「オーディオ」
  6. ファイル形式を「AIFF」に設定
🎙️

ボイスメモ

簡単★★☆

標準アプリで手軽に録音可能

  1. ボイスメモアプリを開く
  2. 楽曲を別のアプリで再生
  3. 赤い録音ボタンをタップ
  4. 録音完了後、停止ボタンをタップ
  5. 録音データを「共有」でAirDropやメールで送信
🔊

Audacity

高機能★★★

本格的な録音・編集が可能（要別アプリ「Audacity Mobile」）

  1. App StoreでAudacity系アプリをダウンロード
  2. 新規プロジェクトを作成
  3. マイク入力設定を確認
  4. 録音ボタンで開始
  5. 録音後「ファイル」→「エクスポート」→「WAV」
iPhone録音のコツ
  • マイク位置：口から20-30cm離して録音
  • 環境：静かな部屋で、カーテンや布団で音を吸収
  • 設定：機内モードにして通知をOFF
  • 音質：GarageBandなら44.1kHz/24bitで自動録音

Android での録音方法

使用率 No.2
📱

RecForge II

おすすめ★★★

Android最高品質・無料版あり

  1. Google PlayからRecForge IIをダウンロード
  2. アプリを開き「録音」をタップ
  3. 設定で「WAV 44.1kHz 16bit」を選択
  4. 楽曲を再生しながら録音開始
  5. 録音後「共有」→「ファイルで共有」
🎤

Smart Recorder

簡単★★☆

シンプルで使いやすい

  1. Smart Recorderをインストール
  2. マイクアイコンをタップ
  3. 設定で最高音質を選択
  4. 録音開始
  5. 録音リストから共有
📻

Samsung ボイスレコーダー

標準★★☆

Samsung端末標準アプリ

  1. ボイスレコーダーアプリを開く
  2. 設定で「高品質」を選択
  3. 録音ボタンをタップ
  4. 録音完了後、共有アイコンをタップ
Android録音の注意点
  • 権限設定：マイクとストレージのアクセス許可が必要
  • 機種差：メーカーによって標準アプリが異なります
  • 音質設定：必ず高音質モードに設定してください
  • 容量確保：録音前にストレージ容量を確認

PC での録音方法

プロ仕様
🎛️

Audacity

おすすめ★★★

無料・多機能・Windows/Mac対応

  1. 公式サイトからAudacityをダウンロード
  2. 「生成」→「トーン」でカラオケ音源準備
  3. 新規トラックを追加（「トラック」→「新規」→「モノラル」）
  4. 録音レベルを調整
  5. 赤い録音ボタンで開始
  6. 「ファイル」→「書き出し」→「WAVで書き出し」
🍎

GarageBand (Mac)

Mac専用★★★

Mac標準・プロ級品質

  1. GarageBandを起動
  2. 「空のプロジェクト」を選択
  3. 「Audio」→「マイク」を選択
  4. カラオケ音源をドラッグ＆ドロップ
  5. 新しいオーディオトラックで録音
  6. 「共有」→「曲をディスクに書き出す」
🎵

本格DAW

上級者★★☆

Cubase、Studio One等（有料）

  1. DAWソフトを起動
  2. 新規プロジェクト作成（44.1kHz/24bit）
  3. オーディオトラックを追加
  4. 入力レベルを調整
  5. 録音開始
  6. WAV形式でエクスポート
PC録音のコツ
  • マイク設定：USB/XLRマイク推奨、内蔵マイクは避ける
  • サンプルレート：44.1kHz/24bit または 48kHz/24bit
  • バッファサイズ：録音時は大きめ（512samples以上）
  • モニタリング：ヘッドホンで音声確認

音源提出方法

必読
提出前のMUST CHECK
やること ✅
  • 頭出し：音声の開始位置を0秒にする
  • ファイル名：内容が分かる名前をつける
  • トラック分け：メイン・ハモリを別ファイルで
  • 音質確認：WAV/AIFF形式で保存
やらないこと ❌
  • ノイズ除去：自己判断での編集は禁止
  • エコー追加：リバーブ等のエフェクト禁止
  • 音量調整：ノーマライズ等の処理禁止
  • MP3圧縮：非可逆圧縮は音質劣化

ファイル命名規則

MIX作業を効率化するため、以下の命名規則に従ってファイル名をつけてください

✅ 良い例：
rec_vo_main1.wav　（メインボーカル パート1）
rec_vo_main2.wav　（メインボーカル パート2・歌いきれなかった部分）
rec_vo_harmony_high.wav　（高音ハモリ）
rec_vo_harmony_low.wav　（低音ハモリ）
rec_vo_chorus.wav　（コーラス部分）
❌ 悪い例：
録音1.wav　（内容が不明）
voice.wav　（どのパートか不明）
untitled.wav　（全く内容が分からない）
命名のルール：
